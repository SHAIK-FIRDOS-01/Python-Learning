# 🐍 My Daily Python Journey

Welcome to my daily Python learning repository! I created this space to hold myself accountable, track my progress, and build a consistent coding habit. I push new code, notes, or projects here every single day.

## 🎯 Goals
* Write and push Python code daily.
* Solidify my understanding of core concepts.
* Transition smoothly into advanced topics like Object-Oriented Programming (OOP), Data Structures and Algorithms (DSA), and eventually Machine Learning and AI.
* Build a personal reference library of algorithms and scripts.

## 📚 What's Inside?
This repository is organized by days/topics. Here is a general overview of what I am covering:

- **Fundamentals:** Syntax, data types, loops, and functions.
- **Object-Oriented Programming (OOP):** Classes, inheritance, and polymorphism.
- **Data Structures & Algorithms (DSA):** Lists, dictionaries, trees, sorting, and searching algorithms.
- **Mini-Projects:** Small scripts and applications to apply what I've learned.

## 🚀 How to Navigate
Each folder represents a specific day or a specific topic. Inside, you will find:
1. The Python script(s) (`.py` files).
2. Brief comments or notes explaining the logic and what I learned that day.

## 💡 Why I'm Doing This
Consistency is the key to mastering any programming language. By making a public commitment to push code daily, I am building muscle memory and a strong foundational understanding of Python.
